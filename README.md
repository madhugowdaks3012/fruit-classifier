# 🍎 Fruit Classifier using Deep Learning

A simple and interactive web application built using **Streamlit** and **TensorFlow/Keras** that classifies fruit images into different categories.

## 🚀 Features

- Upload image from **device**
- Upload image via **URL**
- Classifies fruits into:
  - Apple 🍎
  - Banana 🍌
  - Mango 🥭
  - Orange 🍊
  - Pineapple 🍍
- Displays prediction with **confidence score**
- Clean and user-friendly UI

## 🧠 Model Details

- Model trained using **TensorFlow/Keras**
- Input image resized to **128x128**
- Normalized pixel values (0–1)
- Output layer predicts probabilities for 5 classes

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🌐
- TensorFlow / Keras 🤖
- NumPy
- PIL (Python Imaging Library)
